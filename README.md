# netops2devops blog

My personal blog built with Mkdocs and deployed via Github pages. 

## Instructions

1. Add, remove or update posts in markdown under `docs/`
2. Review updates locally `poetry run mkdocs serve`
3. Commit and push any changes to main
4. Deploy using `poetry run mkdocs gh-deploy --strict`
